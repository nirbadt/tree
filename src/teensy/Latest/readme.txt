This folder includes two Teenzy codes - one for each tree.
The only difference between them is the color of outgoing signal and incoming signal.

To burn this code, download teenzyduino https://www.pjrc.com/teensy/teensyduino.html
This is an addon for ardunio that allows to burn the teenzy.

teenzy board version is Teenzy 3.5
On first use, you need to burn a code and then press reset as appears in the instructions of teenzy website
